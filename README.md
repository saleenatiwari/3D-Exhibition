# 🌸 3D Exhibition — with Blender

A personal Blender learning journal turned into a website. Each page is a stop in the exhibition — a 3D model I made, organized by month, showing progress from the very first attempt to now.

🔗 **Live site:** [blender2-rosy.vercel.app](https://blender2-rosy.vercel.app)

---

## 🖼️ What's inside

The site walks you through my Blender projects chronologically, with Back / Next navigation between each exhibit.

| Month | Piece |
|-------|-------|
| March 2026 | 🫖 First ever Blender artifact — a mug |
| March 2026 | 📌 Cork board with polaroid photos pinned to it |
| April 2026 | 🪑 Draped table with a cloth |
| April 2026 | 🍒 Bowl of cherries render |

---

## ✨ Site features

- **Pink neon cursive landing page** — "Welcome to my 3D Exhibition with Blender"
- **Back / Next navigation** — move through the exhibition one piece at a time
- **Month timestamps** at the bottom of each page to track progress
- **Mixed media** — some pieces are live 3D models, some are renders displayed as images
- **Polaroid cork board** — a whole scene, not just a single object 📌
